This section contains core modules of the application, which are the following:

  • Business Intelligence
  • Patient Search
  • Multi Patients View
  • Patient Summary
  • Allergies
  • Medications
  • Contacts
  • Problems / Diagnosis

This modules are crucial for the application functionality, and should not be removed in order for application to behave properly.
